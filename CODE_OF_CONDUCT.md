# Code Of Conduct

## Our Standard

Be respectful, constructive, and specific. Assume good intent, but take responsibility for impact. Harassment, threats, hateful language, and sustained disruption are not acceptable.

## Scope

This applies to the repository, issues, pull requests, discussions, and project-related spaces.

## Enforcement

Maintainers may edit, hide, or remove comments, close threads, block users, or take other reasonable action to protect the project space.

## Reporting

Use GitHub repository moderation tools or the security contact in SECURITY.md when a report should not be public.
