1. Deterministic inference engine for audio, room, person, and sync health.
2. Real-data fixture suite with expected outputs and regression gates.
3. Audio classification: silence, music, speech, ambient, clipped.
4. Room confidence model: low light, glare, flicker, flat scene, usable plane confidence.
5. Person confidence model: stable, absent, unstable, false-positive risk, multi-subject risk.
6. State taxonomy and actionable runtime messaging.
7. Sync recovery model with retries, host-missing guidance, and confidence.
8. Debug overlay and inspectable reasoning.
9. Reproducible session summaries and export metadata.
10. Performance and determinism checks across all fixtures.
